# 용지 관리 페이지 스크린샷

**URL:** https://sungjin-print100-nagi.vercel.app/admin/db/papers

**배포 완료:** 2026-02-04

## 화면 구성

### 용지 종류 (papers)

- **순서 변경**: ▲▼ 버튼으로 순서 조정
- **이미지 업로드**: + 버튼 클릭하여 이미지 업로드
- **CRUD**: 추가, 수정, 삭제 버튼

**현재 데이터:**

1. 스노우지 (snow) - 스노우 화이트, 고급 용지
2. 인스퍼 (inspirer) - 랑데뷰 인스퍼, 최고급 용지
3. 모조지 (mojo) - 백상지, 일반 용지

### 용지 단가 (paper_costs)

- **34개 단가 데이터** 표시
- 용지, 평량, 기준 용지, 장당 단가, 마진율, 상태, 작업 버튼

**데이터 예시:**

- 모조지 80g (390x270): 14원/장, 마진율 1.5
- 스노우지 120g (467x315): 28원/장, 마진율 1.5
- 인스퍼 240g (467x315): 98원/장, 마진율 1.5

### 데이터 요약

- 용지 종류: 3개
- 단가 데이터: 34개
- 467x315: 17개
- 390x270: 17개
